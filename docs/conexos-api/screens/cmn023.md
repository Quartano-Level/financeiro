## O que esta tela faz
**CFOP** (Código Fiscal de Operações e Prestações) — define como cada operação de entrada/saída é tributada
e contabilizada. Referenciado em todos os itens fiscais (NFs, pedidos) via `cfoEspCod`.

## Como chegar
`home → /cmn023` → **CFOP - Pesquisa** → Editar → `/cmn023#/cadastro/{cfoEspCod}`.

## Layout visual (verificado)
- **Filtros:** Cód. CFOP · Tipo · Descrição CFOP · Tipo Pessoa · Consumidor Final · NF Incentivo Fiscal ·
  Estoque Contábil · Situação (ATIVO) · Operação Drawback · Descrição Oficial.
- **Grid:** Cód. CFOP · Tipo · Descrição CFOP · NF Incentivo Fiscal · Estoque Contábil · Tipo Documento ·
  Gerencia Estoque · Documento Federal · Situação · Resultado Fiscal · Livro Fiscal · Plano Financeiro ·
  Classificador Tributário · Cód. Classificador · Unidade Negócio · COFINS (CST P/...).

## Schema (`CmnCfop`)
`cfoEspCod`, `cfoDesNome`, `cfoVldTipo`, `cfoVldFiscal`, `cfoVldTratFisIpi`/`cfoVldTratFisIcms` (tratamento
fiscal IPI/ICMS), `cfoVldLivroFiscal`, `gerNum` (natureza gerencial), `plaNum` (plano de contas).

## Quirks
Cadastro-mestre fiscal. Toast informa CFOPs nunca utilizados. Fase 00.
