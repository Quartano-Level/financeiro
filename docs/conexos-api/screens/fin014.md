## O que esta tela faz
**Baixa de Títulos a Receber** — recebimento/baixa de títulos do cliente via **borderô**. Espelho do `fin010`
(a pagar) no lado a receber. `taxasBaixa/list` traz as taxas de conversão da baixa; `recebiveis/list` os títulos.
