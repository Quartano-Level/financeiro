## O que esta tela faz
Modal **Informações e Observações** (controller `com131`): textos livres do documento — observações para o
livro fiscal, observações da NF-e e informações adicionais de interesse do fisco. Comum a docs fiscais/financeiros.

## Como chegar
Doc em Edição → rodapé **Mais Ações → Informação/Observação** (permissão `GET /api/permissoes/new/com131`).

## Layout visual (verificado, com298 doc 8665)
- **Abas:** Observações da NF-e (default) · Informações do Documento · Observação do Doc./Complemento. Botão × fecha.
- **Observações da NF-e:** `OBSERVAÇÃO CUSTOMIZADA PARA O LIVRO FISCAL` (textarea) · `OBSERVAÇÃO DA NF-E`
  (textarea) · `INFORMAÇÕES ADICIONAIS DE INTERESSE DO FISCO` (textarea). Cada bloco tem botão + (adicionar).

## Quirks
Sub-tela (modal), sem rota. Controller `com131`. Também acessível pelo Mais Ações de com297/com296.
