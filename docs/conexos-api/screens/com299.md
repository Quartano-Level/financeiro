## O que esta tela faz
**Financeiro a Receber** — documentos a receber do cliente (lado crédito). Contraparte do **Financeiro a
Pagar** (`com298`); compartilha o DTO **`FinDocCab`** (distinção por `docVldTipo`). Inclui adiantamentos de
cliente (`docVldTipoAdto`).
