## O que esta tela faz
**Geração Automática de NF-e** — emissão e **transmissão em lote** de NF-e ao SEFAZ/Siscomex: gera XML,
monta lote, transmite, consulta retorno, imprime DANFE. Estados de cada NF-e (Homologado/Transmissão/Consulta/
Impressão) visíveis no grid. "Ambiente: PRODUÇÃO / Serviço: DISPONÍVEL".

## Como chegar
