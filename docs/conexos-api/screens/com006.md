## O que esta tela faz
**Cadastro de Produtos** — itens negociados, com classificação fiscal **NCM/TEC** e alíquotas de importação
(II/IPI/PIS/COFINS) derivadas do NCM (`imp013`). Referenciado em pedidos e NFs (`prdCod`).

## Como chegar
`home → /com006` → **Cadastro de Produtos - Pesquisa** → Editar → `/com006#/cadastro/{prdCod}`.

## Layout visual (verificado)
- **Filtros:** Produto · Descrição Produto · Cód. Alteração · NCM · SEQ · Cód. Externo · Tipo · Necessidade LI ·
  Registro Vigente (SIM) · Situação (ATIVO) · Cód. Indicador de Operações (Serviços) · Descrição Classificador
  de Serviço.
- **Grid:** Produto · Descrição Produto · Cód. Alteração · NCM · SEQ · Cód. Externo · Tipo · Cód. Exportador ·
  Necessidade LI · Unidade de Medida Sugestiva · Unidade Primária · **Alíquota II/IPI/PIS/COFINS** · Decreto de
  Redução de II · ...

## Schema (`ComProdutos`)
`prdCod`, `prdDesNome`, `prdEspCodExterno`, `tecEspCod` (NCM/TEC), `prdVldTipo`, `prdPreVlrvenda`/`Vldminimo`/
`Vldmaximo`, `undCod`, `prdCodExportador`/`Importador`/`Fabricante`, `prjCod`/`ctpCod`.

## Quirks
Alíquotas tributárias vêm do **NCM/TEC** (`imp013` Cadastro de Mercadorias na TEC). Sub-listas: tipoImportacao,
similarCorrelatos.
