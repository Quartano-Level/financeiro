## O que esta tela faz
**Proforma de Importação** — fatura proforma (pré-fatura) que antecede a **Invoice** de importação (`log009`).
Liga ao pedido (PO), exportador, origem, valor de seguro e contrato. Início documental do embarque (fase 20).

## Como chegar
`home → /log003` → **Cadastro de Proforma - Pesquisa** → Editar.
