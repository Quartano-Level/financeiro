## O que esta tela faz
**Documento financeiro a pagar** (fornecedor exterior, frete, despachante, tributos). Diferente da NF de
saída, aqui a aba **Itens = "Conta do Projeto"** traz os **encargos/impostos** como linhas (lado
entrada/pagamento da conciliação). Compartilha o DTO `FinDocCab` com com296/com299.

## Como chegar
`home → /com298` → Pesquisa → Editar → `/com298#/cadastro/{docCod}` (abas **Documentos** | **Itens**).

## Layout visual
- **Abas:** Documentos | Itens.
- **Itens → grid PRODUTOS:** Ordem · Valor · Projeto · **Conta do Projeto** · Conta · Centro de Custos ·
  Processo · Unidade de Negócio · CFOP · Complemento · Moeda.
- **Documentos:** RESUMO DO DOCUMENTO (Qtd Itens, Valor Bruto/Líquido, Situação=FINALIZADO) · RESUMO DOS
  TÍTULOS (Valor dos Títulos/Pago/Permutado/em Aberto/a Permutar) · Endereço.

## Exemplo real (verificação)
Doc `8665`: aba Itens = COFINS NACIONALIZAÇÃO R$ 2.472.514,10 · IPI · PIS · TAXA DE REGISTRO SISCOMEX
(4 itens, sem linha de frete — frete fica em outro doc a-pagar, do agente de carga).
`POST /api/com298/comDocProdutos/list/8665/1`.

## Quirks
- Itens aqui = **Conta do Projeto** (encargos), ao contrário de com297 (mercadoria).
- `FinDocCab` é o DTO comum a com296/com298/com299 (distinção por `docVldTipo`).
