## O que esta tela faz
Modal **Encargos Gerais** de um documento fiscal: consolida a composição de custo/encargo. A aba **IMPOSTOS**
é a **fonte "ENCARGOS GERAIS > IMPOSTOS"** da conciliação — traz FOB, **FRETE INTERNACIONAL**, SEGURO,
TAXA CE e os tributos (II/IPI/PIS/COFINS/ICMS/Taxa Siscomex) por documento.

## Como chegar
**Não há rota direta.** É sub-tela: abra uma NF (`com297` Fiscais de Saída, ou `com296`/`com298`) em Edição →
rodapé **Mais Ações → Encargos Gerais** → modal abre na aba **IMPOSTOS**.

## Layout visual (marcos para computer-use)
- Modal centralizado, título da aba ativa = **IMPOSTOS** (há outras abas no modal).
- Grid: colunas **Ordem · Encargo · % Alíq. · Valor MN · Valor Dólar**; ícone de refresh no topo-direito;
  checkbox "Exibir Todos os Registros" no rodapé; botão fechar (×) / "Fechar".
- Linhas típicas: 1 VALOR TOTAL NF · 10 FOB · **20 FRETE INTERNACIONAL** · … MARKUP/DESCONTOS/FEE.

## Schema da resposta (`EncargosGeraisDTO`)
| Campo | Conteúdo |
|---|---|
| `impostos[]` (`ComDocProdtribGeral`) | **a grid IMPOSTOS** — `dtrNumOrdem`, `impDesNome`, `dtrPctAliquota`, `dtrMnyValormn`, `dtrMnyValorDolar` |
| `despesas[]` (`ComDocProdutosDespesasGeral`) | despesas por conta (`ctpDesNome`, `dppMnyValorMn`) |
| `totalImpostos[]` | totalização |
| `encargosGerais[]` (`ComDocProdtribEncargosGerais`) | breakdown **por produto** (`vlrFob`, `vlrFreteIntl`, `vlrSeguroIntl`, `vlrIi`…) |
| `resumo` (`EncargosGeraisResumoDTO`) | totais do doc (`fob`, `freteInt`, `seguroInt`, `ii`, `ipi`, `pis`, `cofins`, `icms`…) |
| `totalProdutos` | `dprPreTotalBruto`/`Liquido`, custo médio/gerencial |

**FRETE INTERNACIONAL — 3 leituras:** linha de `impostos[]` com `impDesNome=="FRETE INTERNACIONAL"` (Ordem 20)
→ `dtrMnyValormn`/`dtrMnyValorDolar`; ou `resumo.freteInt`; ou `encargosGerais[].vlrFreteIntl`.

## Exemplo real (verificação)
NF docCod `5998` (filCod 2): `GET /api/com017/encargosGerais/1/5998/2/1/1` → FRETE INTERNACIONAL
R$ 9.583,91 (US$ 1.733,61); FOB R$ 188.007,26; VALOR TOTAL NF R$ 271.040,17.
