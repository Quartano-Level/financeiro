## O que esta tela faz
**PTAX / Cotação da Moeda** — cadastro de **índices/moedas** e suas cotações. É a fonte de cotação cambial
do ERP, usada para reconciliar a **Variação Cambial** (substitui o PTAX do BCB no app). Cotação é global
(sem filCod); filtro de data `intDtaData#LE` suportado.

## Como chegar
`home → /cmn156` → **PTAX - Pesquisa**.

## Layout visual (verificado)
- **Filtros:** Código · Índice · **Sigla** (moeda) · Descrição.
- **Grid:** Código · Índice · Sigla · Descrição.

## Quirks / referência
Detalhe da rota de cotação na memória **`conexos-cmn156-cotacao`**: `indEspIdent===moeCodMneg`, `intFltVenda`
= taxa de venda; mapa `moeCod→indCod`. Usado pela Calculadora de Variação Cambial. Cotação global (sem filCod).
