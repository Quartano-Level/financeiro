## O que esta tela faz
Modal de **configuração fiscal / NF-e** do documento (controller `com300`). Define dados de emissão,
ambiente de NF-e, CT-e referenciado, impressão e cancelamento fiscal. Reachable de qualquer doc fiscal.

## Como chegar
NF em Edição (`com297`/`com296`/`com298`) → rodapé **Mais Ações → Fiscal** → modal **"FISCAL - COM_300"**.

## Layout visual (verificado, proc. 103 / NF 5998)
- **Abas:** Inf. Fiscal (default) · Inf. sobre o Serviço · Inf. (NF-e) · Inf. (Eventos Terceiros). Botão × fecha.
- **Inf. Fiscal:**
  - `INFORMAÇÕES NF-E/ROMANEIO`: Data/Hora Emissão · Data/Hora Saída · ICMS Exonerado (NÃO) · Normal/Filhote (PRINCIPAL).
  - `CT-E REFERENCIADO A NF-E` · `NF DE AJUSTE` (NÃO) · `CHAVE CT-E SUBSTITUTO` · `TIPO CT-E` (NENHUM).
  - `NF-E`: Tipo Ambiente Finalização (PRODUÇÃO) · Envia Duplicata para Nf-e (SIM) · Gera Data Movimento ·
    Gerar Informação Adicional · Gerar Suframa (NÃO) · Tipo de Nota de Débito (NENHUM).
  - `IMPRESSÃO`: Impresso (SIM) · Usuário (DANIEL_ROCHA) · Data · `CANCELAMENTO FISCAL` (Cancelado/Usuário/Data) ·
    `SITUAÇÃO ESPECIAL DE RESPONSABILIDADE`.
