## O que esta tela faz
**Contrato de Câmbio (FX)** — fechamento da moeda para pagar a importação. Traz **taxa de fechamento**
(`imcFltTxFec`) e calcula **variação cambial** (Var. Total / Var. Acumulada) nativamente. Vincula hedge e
FINIMP (financiamento à importação). Fonte primária da Variação Cambial do processo.

## Como chegar
`home → /imp059` → **Contrato de Câmbio - Pesquisa** → Editar → `/imp059#/cadastro/{imcCod}`.

## Layout visual (verificado)
- **Filtros:** Código · Nº de Contrato · Data do Fechamento de/até · Cód. Borderô · Banco · Processo ·
  Situação (EM CADASTRO, FINALIZADO) · Cód. Natureza Operação.
- **Grid:** Código · Nº de Contrato · Data do Fechamento · **Taxa Fechamento** · Cód. Borderô · Banco · Moeda ·
  Valor Moeda Neg. · **Var. Total** · Valor Moeda Nac. · **Var. Acumulada** · Situação · Cód. Natureza.

## Schema (`ImpContratoCambio`)
`imcCod`, `imcNumNumero`, `imcDtaFechamento`/`imcDtaLiquidacao`/`imcDtaDebito`, `moeCod`, **`imcFltTxFec`**
(taxa fechamento), `pesCodExportador`/`Recebedor`, `imcEspNatureza`, `imcVldStatus`, `gerNumBaixa/Ativa/Passiva`.
Sub: `vinculoHedge/list`, `vinculoFinimp/list`.

## Quirks
**Variação cambial nativa** (Var. Total/Acumulada) — reconcilia com a Calculadora de Variação Cambial e com
`log009/fechamentoCambio` (`ImpContratoCambioInv`). Correção histórica: imp059 NÃO é contrato comercial. 78 endpoints.
