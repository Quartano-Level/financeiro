## O que esta tela faz
**Registro da Declaração de Importação (DI)** — a declaração aduaneira que formaliza a importação no
Siscomex. Núcleo do **despacho aduaneiro** (fase 35): traz FOB, datas de chegada/embarque, código de
despacho, conhecimento (House/NIC), valor da mercadoria. Base para nacionalização (`imp230`) e tributos.

## Como chegar
`home → /imp052` → **Registro da Declaração de Importação - Pesquisa** → Editar.
