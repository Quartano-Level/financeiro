## O que esta tela faz
**Licença de Importação (LI) via Siscomex Web** — anuências de órgãos para importar mercadorias controladas.
Cada LI tem nº, data de registro, deferimento e vincula-se ao processo (`priCod`). Integra com Siscomex.
