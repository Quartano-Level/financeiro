## O que esta tela faz
Modal **Logística** (controller `com014`) do documento fiscal: configuração de transporte/frete da NF —
tipo de frete, CT-e, transportadora, intermediador, drawback, reboques e volumes.

## Como chegar
NF em Edição (`com297`) → rodapé **Mais Ações → Logística** (permissão `GET /api/permissoes/new/com014`).

## Layout visual (verificado, NF 5998)
- **Abas:** Logística (default) · Inf. Entrega/Retirada · **Reboques - COM_291** · **Volumes - COM_293**.
- **Logística:**
  - `PEDIDOS/LOGÍSTICA`: Nr. Site Cliente · Nr. Site Sub-Cliente · Sub-Cliente · Descrição · Proprietário ·
    **Tipo Frete** (REMETENTE (CONTRATAÇÃO)) · Via de Transporte · Presença do Comprador (NÃO SE APLICA) ·
    Indicador de Intermediador/Marketplace (OPERAÇÃO SEM INTERMEDIADOR).
  - `INFORMAÇÕES CTE`: UF/Cidade (Início) + Cód. IBGE · UF/Cidade (Fim) + Cód. IBGE.
  - `TRANSPORTADORA` (+) · `INTERMEDIADOR DA TRANSAÇÃO` (+) · `DRAWBACK` (Número do Ato, Filial, Modalidade, Tipo).
