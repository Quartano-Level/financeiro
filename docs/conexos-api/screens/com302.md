## O que esta tela faz
Modal **Docs Relacionados** (controller `com302`): vincula documentos relacionados/retorno à NF e identifica
a **Nota Fiscal Mãe** (NF de origem). 101 endpoints no swagger (controller rico de vínculos documentais).

## Como chegar
NF em Edição (`com297`) → rodapé **Mais Ações → Docs Relacionados** (permissão `GET /api/permissoes/new/com302`).

## Layout visual (verificado, NF 5998)
- **Abas (6, verificado):** Doc. Vinculados - COM_091 (default) · Docs Relacionados · Vínculo dos Itens da
  NF - COM_122 · Docs Relacionados (Serviços) - COM_101 · Doc. Relacionados a Serviços no Exterior - COM_298 ·
  Pagamentos Antecipados.
- **Doc. Vinculados:**
  - `DOCUMENTOS VINCULADOS E RETORNO`: grid Tipo · Código · Número · Pessoa · Descrição · Situação · Vínculo
    (vazio na NF 5998).
  - `DOCUMENTO RETORNO`: E/S · Número Documento · Tipo.
  - `NOTA FISCAL MÃE`: Código · Número NF.
- Botões: Salvar · Fechar.

## Quirks
Sub-tela (modal), sem rota. Controller `com302` (101 endpoints). Embute `com091` (Doc. Vinculados).
