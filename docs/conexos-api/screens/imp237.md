## O que esta tela faz
**Presença de Carga** — registra o **evento aduaneiro** de chegada física da carga ao **recinto alfandegado**
(armazém/porto/aeroporto), vinculado ao **CE-Mercante** (Conhecimento Eletrônico) e ao recinto SRF. É transmitido
ao **Siscomex** (botão Transmitir) e é pré-requisito para o registro da **DI** (`imp019`). Marca o início do
despacho (fase 35).

## Como chegar
`home → /imp237` → **Presença de Carga - Pesquisa**.

## Layout visual (verificado)
- **Filtros:** Filial · Recinto Aduaneiro · Via de Transporte · CE Mercante · Atualiza (SIM) · Situação
  (EM CADASTRO, PRESENÇA DE CARGA).
- **Grid:** Filial · Recinto Aduaneiro · **Código SRF** · Via de Transporte · **CE Mercante** · Data Cadastro ·
  Atualiza · Situação · **Mensagem de Erro** (retorno Siscomex).
- Botões: Adicionar · Atualizar · Excluir · **Consultar** · **Transmitir** (Siscomex).
