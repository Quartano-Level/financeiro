## O que esta tela faz
Modal **Propriedades do Documento** (controller `com016`): metadados de ciclo de vida do documento —
cancelamento, estorno, finalização, situação fiscal e dados de digitação. Comum a todos os docs fiscais/
financeiros (com297/com296/com298).

## Como chegar
Doc em Edição → rodapé **Mais Ações → Propriedades** (permissão `GET /api/permissoes/new/com016`).

## Layout visual (verificado, com298 doc 8665)
- **Abas:** Situação do Documento (default) · Observações · Propriedades · Restrições. Botões: Salvar ·
  Cancelar (DOC) · Fechar.
- **Situação do Documento:**
  - `CANCELAMENTO`: Cancelado (NÃO) · Motivo · Data · Usuário.
  - `ESTORNO`: Data Estorno · Usuário Estorno.
  - `DOCUMENTO ESTORNADO`: Tipo · Código.
  - `FINALIZAÇÃO`: Finalizado (SIM) · Data/Hora Finalização (06/02/2026 09:08:54).
  - `SITUAÇÃO DO DOCUMENTO`: Usuário Digitação (VINICIUS_COELHO) · Data Digitação · Situação Fiscal (NORMAL) ·
    Situação (NENHUM) · Cópia (ORIGINAL).

## Quirks
Sub-tela (modal), sem rota. Controller `com016`. Também acessível pelo Mais Ações de com297/com296 (mesmo modal).
