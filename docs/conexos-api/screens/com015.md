## O que esta tela faz
Modal **Descontos** (controller `com015`) do documento fiscal: desconto comercial concedido ao cliente,
com rateio empresa×cliente e comissionados comex. Fonte do **DESCONTO COMERCIAL** na conciliação.

## Como chegar
NF em Edição (`com297`) → rodapé **Mais Ações → Descontos** → modal (permissão `GET /api/permissoes/new/com015`).

  Pagamento · Cód. SP Gerada.
