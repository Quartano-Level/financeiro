## O que esta tela faz
**Controle de Saldo de Itens na Admissão Temporária** — gerencia o **saldo de itens** importados sob o regime
de **Admissão Temporária** (regime aduaneiro especial: mercadoria entra com tributos suspensos por prazo
determinado, p.ex. para feiras, testes, reparo). Controla quanto de cada item ainda está "disponível" no regime
e quanto já foi **baixado** (nacionalizado/reexportado). É a tela por trás de **`imp021` → Mais Ações → Saldo
Adm. Temporária**.

## Como chegar
**Contextual** (não roteável direto — nav direto `/imp230` renderiza a shell mas emite toast "404" por falta de
contexto). Acessa-se via **processo `imp021` → Mais Ações → Saldo Adm. Temporária** (carrega com o Código D.A).
