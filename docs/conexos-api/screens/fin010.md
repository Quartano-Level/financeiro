## O que esta tela faz
**Baixa de Títulos a Pagar** — quitação/baixa de títulos do fornecedor via **borderô** (agrupa baixas, com
cheque/banco/conta financeira). Registra data de pagamento e **taxa de conversão** (`bxaFltTxconv`) — base
de juros perdidos e variação cambial (ver CLAUDE.md). Contraparte a receber = `fin014`.

## Como chegar
`home → /fin010` → **Baixa de Títulos - a Pagar - Pesquisa** → Editar → `/fin010#/cadastro/{borCod}`.

  Valor Total Conta/Corrente · Situação.

## Quirks
Borderô (`borCod`) agrupa baixas de títulos. Permuta automática (`permutaAutomatica*`). 43 endpoints.
`psq015/baixasTitulo` traz as datas de baixa para relatórios de juros.
